# Interactive Analytics for Smart House Price Valuation
##  Objective (Problem Statement)
* This project aims to analyze the daily power generation dataset using Python to understand generation patterns, identify trends, detect issues affecting production, and provide recommendations for better decision-making.

* The dataset contains daily records of power generation values, which can be used to study performance variations, peak production periods, and possible inefficiencies.

* The Dataset used in this should be cleaned ,transformed and to give the clear visulization.

* The goal of this project is to perform different types of data analysis to gain meaningful insights from the dataset.
  
## **Outcome**
* Understand daily power generation trends
* Identify highest and lowest generation days
* Detect patterns in electricity production
* Find possible reasons for fluctuations
* Predict future generation
* Provide suggestions to improve power production efficiency

## **Domain**
The Domain of this project is energy

#**Dataset Information**

**Source:** India Data Portal /Energy <font color="blue"><u> **https://indiadataportal.com/p/power/r/mop-power_generation-pl-dl-abc**</font></u>

**Year / Timeline:** Data collected during 2017 to 2025

**Dataset Column/features Description:**

* **ID**- It tells us the Unique value in dataset.

* **Date** -Dates of the daily energy consuption.

* **Region**- Region wise energy consuption.
* **State Name**-Names of the state.
* **State Code**-Codes of the state.
* **Sector**-Defines the sector.
* **Station Type**-Type of the stations.
* **Power Station**-Power station names.
* **Power Station Unit**-Unit used by power stations.
* **Monitored Capacity**-Capacity of the power stations.
* **Today Gen Prgm**-Programe of todays power generation.
* **Today Gen Act**-Todays actual power generation.

## Conclusion
The analysis shows that power generation varies across states and stations, with a strong relationship between planned and actual generation. However, there are gaps indicating inefficiencies and underutilization of capacity. Improving monitoring and optimization can help increase overall power generation efficiency.
